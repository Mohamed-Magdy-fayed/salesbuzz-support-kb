# Tags Index

Add tags to incident/runbook/known-error articles under the **Tags** section.

## Common tag prefixes
- `module:` (e.g., `module:auth`, `module:integration`)
- `error:` (e.g., `error:timeout`, `error:http-500`)
- `cause:` (e.g., `cause:misconfig`, `cause:db-lock`)
- `fix:` (e.g., `fix:restart-service`, `fix:update-config`)
